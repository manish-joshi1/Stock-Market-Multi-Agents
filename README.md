# Stock Market Multi-Agent Crew

This project uses CrewAI to analyze a stock with multiple AI agents.

The workflow has two agents:

- **Financial Market Analyst**: fetches live stock market data with `yfinance` and summarizes the stock's current performance.
- **Strategic Stock Trader**: reviews the analysis and recommends whether to **Buy**, **Sell**, or **Hold** the stock.

The app currently runs the crew for `APPLE` from `main.py`.

## Project Structure

```text
.
├── agents/
│   ├── analyst_agent.py      # Analyst agent definition
│   └── trader_agent.py       # Trader agent definition
├── tasks/
│   ├── analyse_task.py       # Stock analysis task
│   └── trade_task.py         # Trading decision task
├── tools/
│   └── stock_research_tool.py # Yahoo Finance stock data tool
├── crew.py                   # CrewAI crew setup
├── main.py                   # App entry point
├── requirements.txt          # Python dependencies
└── README.md
```

## What It Does

1. Loads environment variables from `.env`.
2. Starts a CrewAI crew with an analyst agent and trader agent.
3. Fetches live market information for the selected stock symbol using Yahoo Finance.
4. Produces a stock performance summary.
5. Produces a Buy, Sell, or Hold recommendation.

## Requirements

- Python 3.10+
- A Groq API key, because the agents use:

```python
model="groq/llama-3.3-70b-versatile"
```

## Setup

Create and activate a virtual environment:

```bash
python -m venv env
source env/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file in the project root:

```env
GROQ_API_KEY=your_groq_api_key_here
```

## Run

```bash
python main.py
```

By default, the app runs:

```python
run("APPLE")
```

To analyze another stock, edit `main.py`:

```python
run("TESLA")
```

You can also use ticker symbols such as:

- `AAPL`
- `TSLA`
- `MSFT`
- `GOOGL`
- `NVDA`
